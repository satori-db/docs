Let's get started with satori

1. First, download

   ### Linux

   ```bash
   curl www.satoridb.com/download/linux
   ```

   ### Windows

   ```bash
   curl www.satoridb.com/download/win
   ```
2. Then run satori

   ```bash
   satori -a root admin
   ```
3. Download our SDKs for communicating with the Satori Runtime
