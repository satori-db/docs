Let's get started with satori

1. First, execute

 ## Linux
 ```bash
bash <(curl -s https://satoridb.com/scripts/install_satori_linux.sh)
 ```
## Windows
```bash
wsl && bash <(curl -s https://satoridb.com/scripts/install_satori_windows.sh)
```
 
2. Then run satori

   ```bash
   satori -a root admin
   ```
3. Download our SDKs for communicating with the Satori Runtime
